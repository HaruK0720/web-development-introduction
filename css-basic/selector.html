復習：CSSで最も重要な基本ルール

セレクター { プロパティ：値； }

どこの（セレクター）　何を（プロパティ）　どうする（値）

どこのセレクターのどのプロパティをどういう値にするか

h1 { color: #ff0000; }


idは「#」、classは「.」で指定することができる


4つのセレクターの優先順位

  1. IDセレクター
  2. クラスセレクター
  3. エレメント（要素）セレクター
  4. ユニバーサルセレクター

優先順位が高いものから順に、スタイルが適用される

IDセレクターが一番優先順位が高く、ユニバーサルセレクターが一番優先順位が低い
